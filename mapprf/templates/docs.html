Documentação

O Aplicativo MapPRF - Mapeamento de Ocorrências da Polícia Rodoviária Federal foi desenvolvido no âmbito do 2° Concurso para Dados Abertos do Ministério da Justiça & W3C Brasil. Este concurso aconteceu através de uma parceria estabelecida entre o Ministério da Justiça (Secretaria Executiva e Departamento da Polícia Rodoviária Federal), a Controladoria-Geral da União e o Ministério do Planejamento, Orçamento e Gestão (SLTI) tendo como referência a base de dados do Sistema de Informações Gerenciais (SIGER), mantida pelo Departamento de Polícia Rodoviária Federal.

Para participar do I Concurso Apps.Gov – SBTI 2014 a aplicação sofreu mudanças no layout além de possuir outras features:

  • Filtragem de dados através dos gráficos: Basta clicar no gráfico em um campo com valores que o mesmo fará uma filtragem referente ao campo selecionado. Assim, novas informações são dispostas em todos os gráficos, fornecendo uma maior imersão e melhor capacidade de análise dos dados;
  • Ao imergir nos dados dos gráficos, é possível retornar para os valores anteriores clicando no botão que aparecerá em seguida ao clique;
  • É possível selecionar todas as BRs através de uma busca, no topo superior após ter selecionado a opção “Índice de BRs”;
  • Foi adicionado uma busca geral na página inicial, onde é possível procurar por Cidade, Estado ou País.

Apesar da base de dados disponibilizada ser muito rica e possibilitar a realização de inúmeras análises, por questões de limitação de tempo, para este concurso, optamos por minimizar o escopo e trabalhar apenas com o conjunto de dados dos registros de acidentes nas rodovias federais. Além deste subconjunto de dados de acidentes de transito, foram utilizados dados da malha viária obtida a partir do Ministério dos Transportes e também dados oriundos do IBGE - Instituto Brasileiro de Geografia e Estatística.

Neste momento, no MapPRF estão disponíveis para análise os dados de acidentes registrados durante os anos de 2011, 2012 e primeiro semestre de 2013. Estes dados podem ser analisados através de mapas e gráficos interativos, conforme pode ser visualizado abaixo:

O MapPRF disponibiliza consultas mais detalhadas de cada BR, como exemplo, na imagem abaixo pode ser visualizada a BR386, com o indicador de acidentes por quilômetro, permitindo que o usuário faça a interação com elementos visuais para parametrização e atualização das informações.

Figura 3: BR386 – Dividida por trechos e com indicador de acidentes por Km

Por considerar que o georreferenciamento pode ser de grande importância para a análise de dados e também para implementação de funcionalidades futuras, na fase de integração de dados elaboramos um procedimento que utiliza interpolação para gerar uma coordenada geográfica (Latitude, Longitude) para identificar o local aproximado de um acidente. Este procedimento foi necessário porque pouquíssimos registros eram georreferenciados na base original. Como exemplo, na imagem abaixo (Fig4) podem ser visualizados o local aproximado de acidentes com mortes.

Figura 4: Local aproximado de todos os acidentes com mortes ocorridos na BR386

Outra funcionalidade do MapPRF é a opção de realizar filtragem através dos dados gerados nos gráficos, basta clicar em um valor de qualquer um dos gráficos e realizar a imersão referente aquele valor. Na figura abaixo é possível visualizar os valores dos gráficos alterados após o clique no dia da semana "Quarta-Feira".

Figura 5: Filtragem de gráficos

É possível verificar os índices de ocorrências por municípios e também efetuar a filtragem deles, a figura abaixo ilustra a funcionalidade:

Figura 6: Índice de Municípios

Entre outra feature, existe a possibilidade de efetuar a busca por cidade, estado ou país. O qual retornará nomes que possuem determinada consulta com links que direcionam para o índice de determinada cidade/estado.

Tecnologias

Dentre as tecnologias utilizadas no desenvolvimento, podemos citar as seguintes tecnologias:

  • SGBD Postgresql
  • PostGis - Extensão Espacial para Postgresql
  • HTML
  • JavaScript
  • Django Framework
  • Bootstrap (Front-end Framework)
  • CanvasJS
  • CSS
  • Google Maps API
  • SASS
  • Bower
  • Grunt
